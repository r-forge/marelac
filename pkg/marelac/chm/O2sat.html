O2sat(marelac)R Documentation

Saturation concentration of oxygen in water

Description

Empirical formulae that can be used to compute saturation concentration of oxygen in water in mg L^{-1}

Usage

O2sat(S = 0, T = K - 273.15, K = T + 273.15, NN = 0, method = c("Weiss", "APHA", "Paul"))

Arguments

S salinity in PSU (for method "Weiss" only),
T Temperature in degrees centigrade,
K absolute Temperature in degrees Kelvin,
NN height above sea level (for method "Paul" only),
method formula to be used, see references for correct application.

Details

Method APHA is the standard formula in Limnology, method Weiss possibly the standard formula in marine sciences. The method after Paul is a simple formula fitted on available tables. To avoid confusion between the arguments (S, T, K, NN) it is advisable to use named arguments in general (e.g. O2sat(T=4).

Value

Vector with oxygen saturation concentration in mg/L.

Examples

  O2sat(T = 20)
  T <- seq(0, 30, 0.1)
  plot(T, O2sat(T = T), type = "l", ylim = c(0, 15), lwd=2)
  lines(T, O2sat(S = 0, T = T, method = "Weiss"), col = "yellow", lwd = 2, lty = "dashed")
  lines(T, O2sat(S = 35, T = T, method = "Weiss"), col = "red", lwd = 2)
