g2mol(marelac)R Documentation

Conversion from gram to mol

Description

Converts gram to mol of a chemical species

Usage

g2mol(species)

Arguments

species grams of a chemical species whose weight must be converted to mol

Value

mol/g

Note

This function includes only some biogeochemically active species

Author(s)

Karline Soetaert <k.soetaert@nioo.knaw.nl>

Examples

g2mol("CO3")
g2mol("hCO3")
g2mol("H")
