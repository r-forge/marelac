sal2cl(marelac)R Documentation

Salinity-chlorinity conversion

Description

Estimates the chlorinity concentration based on salinity

Usage

sal2cl(S=35)

Arguments

S salinity

Value

The chlorinity concentration, g/kg

Author(s)

Karline Soetaert <k.soetaert@nioo.knaw.nl>

References

Wooster et al.,1969

Examples

sal2cl(20)
